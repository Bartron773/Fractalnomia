# Contributing to Fractalnomia

Fractalnomia is an open research framework. Contributions that align with its core purpose are welcome.

## What We're Looking For

- Simulations or detection methods that map conditions for human flourishing
- Visualizations that reveal pattern and becoming across scales
- Philosophical or civic frameworks that extend the five-layer roadmap
- Real-world burst data or case studies for the HCBL

## How to Contribute

1. Fork the repository
2. Create a branch: `git checkout -b your-feature-name`
3. Commit your changes with clear messages
4. Open a pull request with a brief description of what you added and why

## Guiding Principles

Contributions should reflect the core axioms:

- Stewardship over optimization
- Becoming over explaining
- Relationship over isolation

If your work helps people or communities become more fully themselves, it belongs here.
